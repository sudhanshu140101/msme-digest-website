# MSME Digest Website

A responsive static website for **MSME Digest** by Mukesh Mohan Gupta, designed to present structured guidance on the **MSMED Act, 2006**, key litigation areas, and useful reference resources

## Live Demo

[View Live Website](https://msmedigest.indiansmechamber.com/)

## Repository

[GitHub Repository](https://github.com/sudhanshu140101/msme-digest-website.git)

## Overview

This project is a single-page website built to showcase the MSME Digest publication with a clean, professional, and informative layout.

It includes:

- Book introduction and value proposition
- MSME legal/litigation challenge highlights
- Coverage of important MSMED Act sections (15–23, including Sections 18 and 19)
- Author profile and credibility details
- Direct links to sample chapter and case index PDFs
- Clear call-to-action for purchase/enquiry

## Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Custom CSS animations and styling**
- **Vanilla JavaScript**
